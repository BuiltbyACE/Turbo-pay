#  Turbo Pay

Turbo Pay is a digital fare collection and transport management system for matatus. It enables conductors to collect fares via cash and M-PESA, track trips offline, and provides SACCOs with real-time analytics through a modern dashboard.

---

## Features

### Conductor App

* Start & manage trips
* Record fares (Cash, M-PESA, QR)
* Offline-first with auto-sync
* Real-time trip summary

### Admin Dashboard

* Revenue analytics & charts
* Trip monitoring
* Payment tracking
* Fleet & route management

---

## Tech Stack

* **Frontend:** Angular + Tailwind CSS
* **Mobile:** Flutter (Dart)
* **Backend:** Django + Django REST Framework
* **Database:** PostgreSQL
* **Payments:** M-PESA (Daraja API)
